# Maximum Submatrix

## Description

Given an `n x n` matrix of `positive` and `negative` integers, find the submatrix with the largest possible sum.

## Example

```text
Given matrix = 
[
[1,3,-1],
[2,3,-2],
[-1,-2,-3]
]
return 9.
Explanation:
the submatrix with the largest possible sum is:
[
[1,2],
[2,3]
]
```
