# Median of K Sorted Arrays

## Description

There are `k` sorted arrays `nums`. Find the median of the given `k` sorted arrays.

## Example

Given nums = `[[1],[2],[3]]`, return `2.00`.

## Solution

一个相对比较直接的方法就是用类似merge k sorted linkedlist的方法来做。
